# Projeto OSSI — Organização Social Santa Isabel

Este projeto apresenta uma análise acadêmica relacionada à Organização Social Santa Isabel, com foco em compreender dificuldades digitais enfrentadas pelo público atendido e propor soluções de apoio tecnológico.

## Objetivo

Identificar necessidades relacionadas ao uso de tecnologia, acesso a serviços digitais, segurança online e orientação prática para pessoas com pouca familiaridade com recursos digitais.

## Principais pontos observados

- Dificuldade no uso de aplicativos bancários e Pix;
- Dúvidas sobre aplicativos do governo;
- Medo de golpes digitais;
- Necessidade de orientações simples e recorrentes;
- Baixa familiaridade com celular e internet.

## Aprendizados

O projeto contribuiu para compreender como a tecnologia pode ser aplicada de forma social, acessível e responsável, respeitando as dificuldades reais dos usuários.
