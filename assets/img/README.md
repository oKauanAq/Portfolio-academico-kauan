# Imagens

Esta pasta será utilizada para armazenar imagens usadas na página do portfólio, como fotos, ícones ou elementos visuais.

## Objetivo

Organizar os arquivos visuais utilizados no portfólio acadêmico e profissional.

## Uso previsto

As imagens aqui armazenadas poderão ser utilizadas no arquivo `index.html` e na personalização visual da página publicada no GitHub Pages.

## Observação

Evitar o uso de imagens pesadas ou sem relação com o objetivo do portfólio.
